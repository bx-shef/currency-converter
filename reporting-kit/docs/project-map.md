# Карта проекта — Конвертер валют (НБ РБ)

> Точка входа в проект для людей и AI-агента.
> На этот файл опираются навыки отчётов (`/report-status`, `/report-questions`).

## Цель

Конвертер валют по официальному курсу Нацбанка РБ: статическое (SSG) Nuxt-приложение,
работающее и как обычный сайт, и как iframe-приложение внутри портала Bitrix24
(панель чата + мобильное контекстное меню сообщения).

## Шаги

| # | Этап | Статус |
| --- | --- | --- |
| 1 | Базовый конвертер по курсам НБ РБ (SSG, кэш, тема) | ✅ |
| 2 | Встройка в Bitrix24 (SDK, install-flow, виджет чата) | ✅ |
| 3 | i18n под языки портала Б24 | ✅ |
| 4 | Деплой: GHCR + Watchtower за nginx-proxy, CSP по sha256 | ✅ |
| 5 | Отчётность по проекту (reporting-kit) | 🟡 |

## Что сделано

- Конвертер: курсы `api.nbrb.by`, кэш в `localStorage` (TTL 12 ч), адаптивный шаг +/−,
  сумма прописью, формула, тема (colorMode b24ui).
- Встройка в Б24: `useB24` поверх `B24Frame`, `/install` биндит два плейсмента
  (`IM_TEXTAREA` + `IMMOBILE_CONTEXT_MENU`) на `/widget/converter`.
- i18n (`@nuxtjs/i18n`), зеркалит языки портала; паритет ключей под тестами.
- CI/CD: lint + typecheck + Vitest (unit + nuxt) + `nuxt generate`, прод-образ
  `nginx-unprivileged`, CSP без `unsafe-inline` (хэши скриптов на сборке).
