## 1. Что это

Конвертер валют по официальному курсу НБ РБ. **Статика (Nuxt 4 SSG)**, без серверной части.
Работает в двух режимах: обычный сайт (standalone) и iframe-приложение внутри портала
Bitrix24 (виджет над полем ввода чата + dual-mode главной). Стек и разбор по файлам —
в [`CLAUDE.md`](../CLAUDE.md).

## 3. Что проверить 🔍

Автотесты не покрывают реальный портал и живые цели — это **ручные** проверки:

- **Портал Б24** (нужен реальный портал): установка → bind плейсмента `IM_TEXTAREA` → виджет
  «Вставить в чат» → dual-mode главной. Полный чек-лист — [`MANUAL_TEST_PLAN.md`](MANUAL_TEST_PLAN.md) §5–7.
- **Цели Метрики** (только standalone, нужен валидный счётчик): [`MANUAL_TEST_PLAN.md`](MANUAL_TEST_PLAN.md) §8.

**Известные ограничения** (портальные квирки — без реального портала Б24 не воспроизвести и не

#138 (виджет не подстраивается под ресайз фрейма до клика).

## 4. Рассмотрено и отклонено (decision-log)

Память, чтобы уже отвергнутое не предлагалось заново.

**Уровни развития** (из плана, который выполнен): **A. Статикой** (0 инфры — всё сделанное выше) →
**B. +1 serverless-функция** (сбор фидбэка в приватный репо) → **C. +сервер (Nitro «Black Hole»,
serverside OTel, очереди)**. Инвариант проекта — **без серверной части**; B/C намеренно его ломают,
поэтому берутся **только под конкретный спрос** и по решению владельца, не по умолчанию.

**Явно НЕ брать** (N/A для статики): BullMQ/Redis/Postgres/worker-fleet · токен-стор · AI-агент ·
edge-security middleware (есть nginx) · role-guides (нет server/admin-ролей).

**Отклонено/покрыто осознанно:**
- **P1-2 (рейтинг-nudge на Маркет)** — не берём: страница уже с 👍/👎-nudge + двумя промо-карточками;
  ещё один nudge = пере-наджинг, вредит основному сценарию. Только по явному запросу владельца.
- **P2-5 (свободный фидбэк-форма)** — покрыто: канал уже есть — `CustomDevCard` → `offer.bx-shef.by/#brief`
  + 👍/👎-сигнал. Отдельную форму/`mailto` не плодим.
