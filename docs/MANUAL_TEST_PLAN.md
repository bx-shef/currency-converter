# План ручного тестирования

> Last reviewed: 2026-07-23

Чек-лист перед релизом/публикацией в Маркет. Покрывает то, что **нельзя** проверить
автотестами: реальный портал Bitrix24 (install-flow, плейсмент, вставка в чат),
живые курсы НБ РБ, темы, мобильную раскладку и телеметрию. Автотесты (`pnpm check`)
и визуальный харнесс (`pnpm screenshots`, см. [`VISUAL_VERIFICATION.md`](VISUAL_VERIFICATION.md))
идут **до** этого плана — здесь только ручные сценарии.

Как пользоваться: пройти сценарии в колонке «Шаги», сверить с «Ожидание», отметить `✓`/`✗`.
Обязательный прогон — после major-бампа `@bitrix24/b24jssdk` (рантайм SDK автотесты не покрывают).

Легенда: **[S]** standalone (обычный сайт) · **[B24]** внутри портала (iframe) · **[M]** мобильный.
Кнопки копирования скрыты **только в мобильном приложении Б24** (WebView без Clipboard API) —
в standalone и в десктоп-iframe портала они работают.

**Критично для релиза** (проверять в первую очередь, блокеры публикации): 1.1 загрузка курсов,
1.2 пересчёт, 5.2 вставка в чат, 6.1 установка, 9.1 доступность прода. Остальное — важное, но
не блокирующее.

## 8. Телеметрия (Яндекс.Метрика) — только standalone

Внутри портала Метрика заглушена (`public/metrika.js` глушит себя в iframe) — цели **не летят**.
Проверять в standalone с валидным счётчиком; смотреть достижение целей в отчёте Метрики.

| # | Цель | Как вызвать | ✓/✗ |
|---|------|-------------|-----|
| 8.1 | `rates_load_failed` | Открыть `/` при недоступном `api.nbrb.by` | |
| 8.2 | `rates_monthly_missing` | Дневной фид жив, месячный упал | |
| 8.3 | `converter_helpful_yes` / `_no` | Клик 👍/👎 в блоке «Помог курс?» | |
| 8.4 | `market_card_click` / `market_qr_reveal` | Клик CTA карточки Маркета / показ QR | |
| 8.5 | `custom_dev_click` / `custom_dev_qr_reveal` | Клик CTA custom-dev / показ QR | |
| 8.6 | `web_vitals_<lcp\|cls\|inp>_<good\|ni\|poor>` | Открыть `/` standalone, взаимодействовать (для INP), закрыть/скрыть вкладку → смотреть цели в отчёте Метрики | |

Инвариант приватности: цели несут **только факт** (shape/outcome), без сумм/курсов/значений.
